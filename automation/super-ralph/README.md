# Origin v2 Super Ralph Automation

Prompt-driven autonomous implementation runner using `super-ralph` components on `smithers-orchestrator`.

## Commands

```bash
bun install
bun run run
bun run run -- ./input/campaign-01-core-foundation.md
bun run resume -- <run-id>
bun run monitor
bun run typecheck
```
